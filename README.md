##  需要下载的开发环境：

### 后端需要：

IntelliJ IDEA  版本不限  （类似于Eclipse）

JDK 1.8以上

git 最新版即可  （用于网上代码）

maven 3.0.0以上   （用于下载jar包依赖）

### 安卓端需要：

android studio 最新版即可  （用来打开安卓代码）

安卓端参考 《android 第一行代码》 这本书的最后一部分，天气APP的制作过程即可
